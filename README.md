# GalaxyShooter
A small 2D project.

-------------------------------------------------------------------------------
# Controls

W A S D or Arrows to move the ship.

Space for fire.

-------------------------------------------------------------------------------
# Features

- 3 PowersUp

- Cinematic movement, no physics.

- Lives and score counter.

- Game loop.

- A small post-processing treatment.
